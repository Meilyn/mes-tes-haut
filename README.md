##Projet

Application Météo créé avec VueJS composé d'un input de recherche pour la ville.

## Langages utilisés

- HTML5&CSS3 (GRID)
- VueJS

## Build Setup

```
# install dependencies
npm install

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
